# Practica adicional - Modulo 2: Tipos y Comparaciones

Ejercicios de practica adicional que repasan todos los temas del modulo: operadores de comparacion con numeros y cadenas, tipos de datos y conversion de tipos.

---

## Ejercicios

| Archivo | Concepto | Descripcion |
|---|---|---|
| practica_01_mayor_que.py | Operador `>` | print(100 > 1) -> True |
| practica_02_mayor_igual.py | Operador `>=` | print(11 >= 11) -> True |
| practica_03_pass_score.py | Almacenar comparacion | pass = score >= min_score |
| practica_04_pass_grade.py | Almacenar comparacion | pass_grade = score >= min_score |
| practica_05_copy_submission.py | Operador `!=` con cadenas | copy != submission -> True |
| practica_06_nina.py | Operador `!=` con cadenas iguales | "Nina" != "Nina" -> False |
| practica_07_cinco_menor.py | Operador `<` | 5 < 5 -> False |
| practica_08_result_101.py | Operacion aritmetica | result = 100 + 1 |
| practica_09_entero.py | Tipo entero | score = 2 |
| practica_10_cadena.py | Tipo cadena | last_name = "Clooney" |
| practica_11_float.py | Tipo flotante | length = 44.34 |
| practica_12_type_bool.py | Funcion type() | type(False) -> bool |
| practica_13_int_float.py | Funcion int() | int(5.7) -> 5 |
| practica_14_tipo_float.py | Identificar tipo | 23.4 es float |
