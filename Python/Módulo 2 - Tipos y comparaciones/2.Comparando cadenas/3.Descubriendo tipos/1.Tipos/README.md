# Aprendizaje en Mimo

Repositorio de documentacion del aprendizaje en la plataforma Mimo. Cada carpeta corresponde a un bloque de contenido dentro de un modulo. Los archivos `.py` contienen los ejercicios practicados y los archivos `.md` documentan los conceptos aprendidos.

---

## Estructura de carpetas

Cada carpeta de bloque contiene:

- Un archivo `README.md` con los conceptos del bloque
- Archivos `.py` numerados secuencialmente con los ejercicios practicados

```
mimo-python/
├── README.md
├── modulo-01-python-basico/
├── modulo-01-usando-variables/
├── modulo-01-verdadero-y-falso/
├── modulo-02-comparando-numeros/
├── modulo-02-comparando-cadenas/
└── modulo-02-descubriendo-tipos/
```
