# Bloque 01: Autoasignación y Operadores

## Tema 1: Operador de asignación

### Concepto general

La autoasignación es una herramienta poderosa para construir programas.
Este tema introduce los operadores que permiten modificar el valor de una
variable de forma más rápida, sin tener que reescribir su nombre dos veces.

Ya sabíamos que podíamos sumar un valor a una variable escribiendo su
nombre en ambos lados de la asignación, por ejemplo:

```python
likes = 5
likes = likes + 1
```

En lugar de eso, podemos usar el operador `+=` para lograr el mismo
resultado de forma más simple:

```python
sales = 5
sales += 1
```

De manera equivalente, para restar del valor de una variable usamos el
operador `-=`:

```python
sales = 5
sales -= 3
```

### Conceptos reforzados sin código (preguntas de opción múltiple)

- **Qué operador suma al valor de una variable**: el operador `+=`.
- **Qué operador usamos para restar un número del valor de una variable**:
  el operador `-=`.
