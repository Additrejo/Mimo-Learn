## Conceptos reforzados sin código (preguntas de opción múltiple)

- **Por qué la declaración else no tiene su propia condición**: porque su
  condición es la opuesta a la condición de la declaración `if` a la que
  pertenece.
- **Qué operador se usa para ejecutar código solo cuando todas las
  condiciones son True**: el operador `and`.
- **Qué sucede cuando todas las condiciones vinculadas con or son False**:
  el bloque de código se omite.
- **Cuándo se ejecutaría un bloque de código si las condiciones están
  vinculadas con or**: cuando hay al menos una condición que es `True`.
- **Cuándo se omitiría un bloque de código si las condiciones están
  vinculadas con or**: cuando todas las condiciones son `False`.
- **Cuántas condiciones alternativas se pueden usar con los operadores
  and / or**: tantas como se quieran.
