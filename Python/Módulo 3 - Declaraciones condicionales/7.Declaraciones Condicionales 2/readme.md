# Bloque 07: Declaraciones Condicionales 2 (Práctica)

Este bloque corresponde a una sección de práctica que repasa los temas vistos
anteriormente en el módulo: declaraciones `if`/`else`, `elif`, los operadores
`and` y `or`, y errores comunes de sintaxis e indentación.

## Ejercicios de código

1. **ejercicio_19_practica_error_sintaxis_else.py**
   Identifica el error en un código donde la declaración `else` aparece antes
   que la declaración `if` a la que pertenece, generando un `SyntaxError`.

2. **ejercicio_20_practica_sugerencias_amigos.py**
   Predicción de salida con `if`/`else`: al no cumplirse la condición del `if`,
   se ejecuta el bloque `else`.

3. **ejercicio_21_practica_eleccion_postre.py**
   Predicción de salida con `if`/`else` comparando cadenas de texto.

4. **ejercicio_22_practica_mad_meter.py**
   Se elige un valor para una variable de modo que se ejecute el bloque `else`
   en lugar del bloque `if`.

5. **ejercicio_23_practica_indentation_error_colores.py**
   Comparación de dos bloques `if` con distinta indentación para identificar
   cuál de ellos provoca un `IndentationError` y por qué.

6. **ejercicio_24_practica_temperatura_elif.py**
   Predicción de salida con una cadena `if`/`elif`/`elif`.

7. **ejercicio_25_practica_uso_app_elif.py**
   Se agrega una nueva declaración `elif` a una cadena existente.

8. **ejercicio_26_practica_mensaje_idioma_elif.py**
   Se completa la condición de una declaración `elif` para que coincida con
   un idioma específico.

9. **ejercicio_27_practica_bienvenida_email_elif.py**
   Se codifica la palabra clave `elif` en una cadena condicional.

10. **ejercicio_28_practica_or_mensaje_no_leido.py**
    Predicción de salida usando el operador `or`.

11. **ejercicio_29_practica_or_recomendacion_libro.py**
    Se construye una cadena de condiciones alternativas vinculadas con `or`.

12. **ejercicio_30_practica_contar_condiciones_or.py**
    Se identifica cuántas condiciones alternativas están vinculadas con `or`
    en un mismo bloque `if`.

## Conceptos reforzados sin código (preguntas de opción múltiple)

- **Por qué la declaración else no tiene su propia condición**: porque su
  condición es la opuesta a la condición de la declaración `if` a la que
  pertenece.
- **Qué operador se usa para ejecutar código solo cuando todas las
  condiciones son True**: el operador `and`.
- **Qué sucede cuando todas las condiciones vinculadas con or son False**:
  el bloque de código se omite.
