# Modulo 01 - Practica: Python Basico

Curso: Python | Modulo: 1. Python Basico

---

## Descripcion

Esta seccion contiene los ejercicios de la practica general del modulo "Python Basico" en Mimo. Los ejercicios repasan de forma integrada los conceptos cubiertos a lo largo del modulo: nombres de variables, asignacion y actualizacion de valores, tipos de datos, operadores booleanos y f-strings.

---

## Conceptos repasados

- Convencion snake_case para nombres de variables
- Asignacion de valores de cadena, numeros y booleanos
- Actualizacion del valor de una variable
- Concatenacion de cadenas con `+`
- Operador de negacion `not`
- Operador de desigualdad `!=`
- Cadenas formateadas (f-strings) con variables entre llaves
