# Modulo 01 - Practica 2: Python Basico

Curso: Python | Modulo: 1. Python Basico

---

## Descripcion

Segundo bloque de ejercicios de practica del modulo "Python Basico" en Mimo. Los ejercicios integran conceptos de variables, operaciones aritmeticas, concatenacion de cadenas y actualizacion de valores, con enfasis en escenarios aplicados y casos de error.

---

## Conceptos repasados

- Actualizacion del valor de una variable (incluyendo cambio de tipo)
- Asignar el valor de una variable a otra variable
- Concatenacion de cadenas con `+` usando variables
- Division y resultado de tipo float
- Expresiones aritmeticas con parentesis
- NameError al referenciar una variable no definida

---
