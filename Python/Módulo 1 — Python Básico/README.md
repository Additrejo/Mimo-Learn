##  Creando variables

### Nombres de variables

Las variables sirven para que los programas **recuerden información**. Son como cajas de mudanza: tienen contenido y un nombre que dice qué hay dentro.

**Reglas para nombrar variables:**

- El nombre debe ser una **sola palabra** — sin espacios.
- Si se necesitan varias palabras, se usa **snake_case**: conectar las palabras con guión bajo `_`.
- Los nombres deben ser **descriptivos** para entender qué guardan.

```python
#  Correcto — una palabra
city

#  Correcto — snake_case para múltiples palabras
home_city
home_city_province
high_score
friend_request
total_cash_amount

#  Incorrecto — tiene espacio
high score
```

>  Usar `city` o `population` en lugar de `c` o `p` nos ayuda a entender qué hay dentro de la variable.
