## El signo igual

Despues de nombrar una variable, usamos el signo `=` para almacenar un valor dentro de ella.

```python
city = "Miami"
```

La estructura siempre es la misma: `nombre = valor`.

Almacenar un valor en una variable es como poner cosas dentro de una caja etiquetada. La variable `device_type` almacena el valor `"apple"`:

```python
device_type = "apple"
```

Las variables pueden contener todo tipo de valores. Aqui un ejemplo con texto:

```python
car_1 = "Porsche"
```

---

## Strings

Los valores de texto se llaman **strings**. Un string es texto entre comillas dobles.

```python
city = "Miami"
```

Las cadenas pueden contener todo tipo de letras y simbolos, incluidos espacios:

```python
movie_quote = "Winter is coming."
```

Como saber que un valor es una cadena: es un texto entre comillas dobles, como `"Amy"`.

Diferencia entre nombre y valor en una variable:

```python
# name  → nombre de la variable
# "Karin" → valor de la variable
name = "Karin"
```

Que esta pasando en este codigo:

```python
song_name = "Hey Ya!"
# La variable song_name almacena el valor "Hey Ya!"
```
